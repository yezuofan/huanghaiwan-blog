## 三、3种使用方式

### 方式1：Custom Mode（推荐，精确控制）

点击 "Create" → "Custom Mode"，输入：

```
[Style of Music]  // 音乐风格描述
[Subject/Story]   // 歌曲主题或歌词故事
```

**例子1（中文流行）：**
```
中文说唱，深夜独白，讲述北漂青年的奋斗与孤独，节拍强劲，Hook动人
```

**例子2（英文流行）：**
```
Upbeat pop song, love story, summer vibes, catchy chorus, Taylor Swift style
```

**例子3（纯音乐）：**
```
Lo-fi chill hop, rainy cafe background music, mellow piano, relaxed beats
```

### 方式2：Quick Mode（简单模式）

不用写歌词，只要描述想要的音乐风格和感觉，Suno会自动填词。

```
A chill lo-fi track for studying, Japanese city pop style
```

### 方式3：Extend（续写/混音）

上传已有的音频片段，让Suno继续创作或生成伴奏。

## 五、5种热门音乐风格实战教程

### 1. 短视频爆款BGM（最容易火）

**特点：** 15-60秒，洗脑Hook，前3秒就要抓住耳朵

**Prompt模板：**
```
[风格：电子/Duet/House/中文抖音神曲],
节奏感强，[Hook要洗脑/副歌要上头],
适合[场景：跳舞/日常vlog/美食/旅游],
[情绪：欢快/治愈/热血], [时长：30秒循环]
```

**实战例子：**
```
欢快的中文抖音BGM，副歌超级上头，适合舞蹈视频，节奏快，有笛子元素，15秒循环版
```

### 2. 个人情感单曲（最走心）

把自己的故事写成歌，这是Suno最动人的用法之一。

**Prompt模板：**
```
[语言] [风格] 歌曲，关于[你的故事/情感]，
歌词要[真实/感人/有细节]，
[歌手风格参考]，
[速度：慢速抒情/中速叙事/快速激烈]
```

**例子（写给暗恋对象）：**
```
中文民谣，关于暗恋一个人三年却不敢表白的心情，
歌词有具体场景（教室/操场/下雨天），
吉他伴奏，简短有力，类似赵雷风格
```

### 3. 多语言国际化歌曲

Suno支持50+语言，一首歌可以无缝切换语言。

**Prompt模板：**
```
[语言A] 主歌 + [语言B] 副歌，关于[主题]，
[风格]，[情绪]，
[语言切换时机：主歌到副歌/说唱部分换语言]
```

**例子（中英双语）：**
```
主歌中文说唱 + 副歌英文流行，关于创业的艰辛与梦想，
电子+嘻哈混搭，节奏强劲
```

### 4. 品牌/企业主题曲

企业宣传、活动BGM，定制化更强。

**Prompt模板：**
```
[品牌/产品名] 主题曲，[风格：积极向上/科技感/温馨]，
[时长：30秒/1分钟/2分钟]，
[用途：宣传片/活动背景/APP开屏]，
[参考风格：高端品牌音乐/科技感电子]
```

### 5. 影视/游戏配乐

纯音乐场景，适合视频创作者。

**Prompt模板：**
```
[类型：Epic/Chill/Action/Horror] [场景配乐]，
[乐器：钢琴+管弦乐/合成器/吉他]，
[情绪：紧张/舒缓/史诗感/悬疑]，
[速度：渐强/平稳/快节奏]，
[时长]，[无缝循环版/有结尾版]
```

## 六、进阶技巧（用过5首以上再来看）

### 1. 结构化提示词法

把提示词拆成结构化模块，效果远超随机描述：

```
Genre: [具体音乐风格]
Mood: [情绪关键词1, 关键词2, 关键词3]
Theme: [歌曲核心主题]
Lyrics Style: [歌词风格：叙事/诗意/口语化]
Instrumentation: [主要乐器]
Tempo: [BPM或快/中/慢]
Vocal Style: [歌手音色：男声/女声/童声/合唱]
Reference: [参考艺术家或歌曲]
```

### 2. Negative Prompt（反向提示词）

告诉Suno **不要** 什么，同样重要：

```
不要[不想要的元素：太吵/歌词太直白/副歌不够上头/节奏太慢]
```

### 3. 分段生成法

先生成纯音乐版本，再生成人声版本，最后混音。质量比一次性生成更高。

### 4. 善用Remix和Extend

对生成结果不满意？点击 "Remix" 让Suno在原有基础上变化，或用 "Extend" 延长片段。

### 5. 歌词优化技巧

- **具体 > 抽象**："讲述地铁末班车" 比 "关于城市孤独" 更有效
- **情绪词精准**："心如刀割" 比 "难过" 更能触发正确情绪
- **场景细节**："暴雨天躲在便利店门口" 比 "分手" 更动人
