## ChatGPT 为什么国内用不了？

ChatGPT 由 OpenAI 开发，官网 `chatgpt.com` 和 API 服务未对中国大陆开放。国内直连会出现：

- ❌ 浏览器：**连接被重置** / **无法访问此网站**
- ❌ 移动端 A**请检查网络连接**
- ❌ API：`Connection error` / `Request timed out`

**不只是 ChatGPT 这样**——Claude、Gemini、Perplexity 所有海外 AI 服务都面临同样问题。你需要一条通往海外服务器的稳定通道。

## 方案二：API 中转服务（✅ 可选）

### 原理

不直接访问 ChatGPT 网页，而是通过第三方购买 OpenAI API 的调用额度。你在中转平台充值后，用 API Key 对接第三方客户端（如 ChatBox、LobeChat）。

### 操作步骤

1. 找一家 API 中转服务商（搜索"OpenAI API 中转"）
2. 注册账号，充值（通常 ¥10 起）
3. 获取 API Key 和代理地址
4. 在第三方客户端中填写 API Key
5. 开始使用 ChatGPT 的 API 能力

### 优缺点

| 优点 | 缺点 |
|:----|:----|
| ✅ 无需配置网络，直接调 API | ❌ 只能用 API 模式，不能访问 ChatGPT 网页版 |
| ✅ 按量计费，用量少很便宜 | ❌ 中转商良莠不齐，有跑路风险 |
| ✅ 可以用第三方客户端（界面可能更好） | ❌ 需要折腾客户端配置，门槛较高 |

### 适合人群

- 开发者或技术人员
- 主要通过 API 使用 AI（如 Cursor、VS Code 插件）
- 不想装代理客户端的轻度用户

> ⚠️ 注意：API 中转 ≠ ChatGPT Plus/Pro 订阅。中转商卖的是调用额度，部分功能（如 DALL·E 生图、GPTs）可能不可用。

## 方案三：第三方镜像站（⚠️ 临时用）

### 原理

有人把 ChatGPT 的界面"搬"到国内可访问的服务器上。你直接访问这个镜像站，就能像用 ChatGPT 一样使用。

### 优缺点

| 优点 | 缺点 |
|:----|:----|
| ✅ 打开即用，零配置 | ❌ 极易被封禁，经常换域名 |
| ✅ 很多免费或很便宜 | ❌ 数据安全存疑（镜像站能看到你的对话） |
| | ❌ 功能不全（落后官方版本） |
| | ❌ ChatGPT 官方封号风险 |

### 能用吗？

**应急可以，日常不推荐。** 原因很简单：

1. **隐私问题**——你的所有对话记录镜像站所有者都能看到，包括粘贴的代码、文档、个人信息
2. **稳定性差**——镜像站域名频繁更换，今天能用明天可能就挂了
3. **体验打折**——很多镜像站用的是老模型，不支持文件上传、长上下文等新功能

> 如果你只是临时体验一下，搜"ChatGPT 国内镜像"能找到。但长期使用强烈建议方案一或二。

## 常见问题（FAQ）

### Q1：ChatGPT 在中国能用吗？

可以，通过机场或 API 中转都能正常使用。ChatGPT 官方没有封禁中国 IP，只要你从境外节点访问就能注册和使用。

### Q2：用 ChatGPT 会被封号吗？

正常使用不会。OpenAI 封号主要针对：滥用 API、批量注册、违反使用政策的用户。通过正规机场正常对话完全没问题。

### Q3：机场能用来注册 ChatGPT 吗？

可以。注册 ChatGPT 需要一个海外手机号（接码平台可解决），机场提供稳定的境外 IP 完成注册流程。

### Q4：ChatGPT Plus 和中国信用卡

OpenAI 目前不接受中国大陆发行的信用卡。可以在虚拟卡平台（如 Depay/OneKey）充值美元后绑定，或走 iOS 内购（相对方便）。

### Q5：免费的 ChatGPT 方法有吗？

镜像站可以免费体验，但上文说过安全风险。长期免费的话，微软 Copilot（基于 GPT-4）国内可以直接用，功能上可以替代大部分 ChatGPT 需求。

### Q6：ChatGPT、Claude、Gemini 哪个好？

| AI助手 | 强项 | 国内访问方式 |
|:------|:----|:-----------|
| ChatGPT | 全能型，写代码/翻译/创意 | 机场 + API中转 |
| Claude | 长文档/论文分析/编程 | 机场 + API中转 |
| Gemini | 免费量大，配合 Google 生态 | 机场最佳 |

三者各有千秋，**有条件的建议三个都试试**，用机场方案一个客户端全通。
