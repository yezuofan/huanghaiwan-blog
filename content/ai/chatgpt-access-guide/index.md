## 为什么 ChatGPT 国内用不了？

ChatGPT 由 OpenAI 开发，`chatgpt.com` 未对中国大陆开放。直连会遇到：
- 浏览器：连接被重置
- API：Connection error / Timeout
- 移动端 App：网络错误

**不是 ChatGPT 的问题**——Claude、Gemini、Perplexity 所有海外 AI 服务都一样。你需要一条通往海外服务器的稳定通道。

## 方案二：API 中转服务

### 怎么做

找一家 OpenAI API 中转平台 → 充值（¥10 起）→ 获取 API Key → 在第三方客户端（ChatBox/LobeChat）配置 → 开始用。

### 优缺点

| 优点 | 缺点 |
|:----|:----|
| 不用配网络，直接调 API | 只能用 API，不能访问 ChatGPT 网页版 |
| 按量计费，用得少就便宜 | 中转商有跑路风险 |
| 可用第三方客户端 | 配置门槛比网络工具高 |

### 适合人群

开发者、主要通过 API 使用 AI（Cursor 插件等）。

> ⚠️ 注意：API 中转 ≠ ChatGPT Plus 订阅。部分功能（DALL·E、GPTs）可能不可用。

## FAQ

**Q：ChatGPT 在中国能用吗？**
可以。通过网络工具从境外节点访问就能正常使用和注册。

**Q：用 ChatGPT 会被封号吗？**
正常使用不会。封号主要针对滥用 API、批量注册的用户。

**Q：能用来注册 ChatGPT 吗？**
可以。注册需要海外手机号（接码平台），网络工具提供稳定境外 IP。

**Q：ChatGPT Plus 怎么支付？**
中国大陆信用卡不可用。推荐用虚拟卡（如 OneKey）或 iOS 内购。

**Q：ChatGPT vs Claude vs Gemini？**

| AI | 强项 | 国内访问 |
|:---|:----|:--------|
| ChatGPT | 全能型，写代码/翻译 | 网络工具 + API |
| Claude | 长文档/论文分析 | 网络工具 + API |
| Gemini | 免费量大 | 网络工具最佳 |
