## 一、Gemini 是什么？

Google Gemini 是 Google 发布的下一代 AI 模型，发布于 2023 年 12 月。名称沿用了 Google 内部沿用多年的项目代号。

它的核心定位是 **Google 自家的 ChatGPT 竞品**，最大的差异化在于和 Google 生态的深度整合：

| 对比维度 | ChatGPT | Claude | Gemini |
|:--------|:--------|:-------|:-------|
| 开发商 | OpenAI | Anthropic | Google |
| 实时联网搜索 | 需插件 | 需插件 | **原生集成** |
| Google 全家桶 | 无 | 无 | **Gmail/Drive/YouTube 直接调用** |
| 多模态 | GPT-4V | Claude 3 | **Gemini 1.5 Pro 原生** |
| 免费额度 | GPT-3.5 免费 | Claude 3 免费 | **1.5 Pro 免费额最大** |

## 二、谁适合用 Gemini？

### 强场景（Gemini 最擅长的）

**1. Google 全家桶用户**
- 写邮件时直接读 Gmail 收件箱
- 整理 Drive 文件、总结文档
- 查日历、设提醒、管任务
- 如果你每天用 Gmail + Google Calendar，Gemini 比任何 AI 都更深度集成

**2. 长文档/长视频分析**
- Gemini 1.5 Pro 支持 **100 万 token 超长上下文**
- 一本书、一份财报、一集 1 小时 YouTube 视频，直接丢给它分析
- 相当于 ChatGPT Advanced Data Analysis 的超长版

**3. 多语言场景**
- Gemini 对中文的支持比 GPT-4 更好
- 翻译质量稳定，适合跨境电商从业者

### 弱场景（Gemini 不擅长的）

| 场景 | 原因 |
|:----|:-----|
| 严肃编程/代码任务 | Claude 3.5 Sonnet 代码能力更强 |
| 深度推理/哲学讨论 | Claude 逻辑更强 |
| 创意写作 | ChatGPT/Claude 表达更自然 |
| 即时新闻查询 | Perplexity 实时性更好 |

**三者都用：** 不同任务切换，这是重度 AI 用户的常见配置

## 四、10 个实测好用的提示词模板

以下模板均基于 Gemini 1.5 Pro 实测，直接复制使用。

### 1. 邮件处理（读 + 写）

```
我的收件箱里有这周客户的所有邮件，帮我：
1. 标记需要紧急处理的（截止日期已过或客户在等回复）
2. 总结每个重要客户的主要诉求
3. 给每个重要客户起草一封回复邮件
```

### 2. YouTube 视频摘要

```
帮我分析这个视频的核心观点和重要细节：
https://youtube.com/watch?v=XXX
请用结构化格式输出：主题、关键观点、值得关注的细节、我的收获
```

### 3. 文档深度分析

```
我上传了一份 [行业报告/论文/合同]，请：
1. 用 3 句话总结核心结论
2. 列出 5 个最重要的数据点
3. 指出 3 个可能的质疑角度
4. 这份文档对我的 [具体工作场景] 有什么直接帮助？
```

### 4. 日程冲突解决

```
我的日历这周有以下冲突：
[粘贴日历内容]
下周三下午 2 点有个重要客户会议，请帮我重新安排：
- 优先保留下周三 14:00
- 找出其他可以挪动的会议
- 给每个被挪动的会议相关人员起草改期邮件
```

### 5. 数据对比分析

```
我有两个产品的数据报表，请对比分析：
产品A：[粘贴数据]
产品B：[粘贴数据]

请从以下维度对比：
1. 增长趋势（哪个增速更快）
2. 用户留存差异及可能原因
3. 最关键的差异指标
4. 各产品的最大风险点
```

### 6. 代码审查

```
请审查以下代码，指出：
1. 潜在 Bug 和安全问题
2. 性能优化建议
3. 代码风格改进建议
4. 添加什么测试用例

代码：[粘贴代码]
语言：[语言类型]
```

### 7. 中英翻译（商务级）

```
请将以下中文内容翻译成专业商务英文，要求：
1. 符合西方商业文化表达习惯
2. 不要直译，要意译
3. 保持专业语气
4. 标注文化差异需要注意的点

[中文内容]
```

### 8. 决策分析

```
我面临以下决策困境：
[描述你的选项和背景]

请用以下框架帮我分析：
1. 每个选项的优势/劣势
2. 潜在风险（最坏情况是什么）
3. 后悔最小化选择（如果错了哪个后悔最小）
4. 下一步行动建议
```

### 9. 学习计划制定

```
我想在 [时间周期，如：8周] 内学会 [技能名称]，目前水平是 [起点]。
请制定：
1. 每周学习目标（具体可操作）
2. 每天投入 [X] 小时的学习安排
3. 关键里程碑和检验方法
4. 学习资源推荐（免费优先）

### 10. 报告快速起草

```

要求：
1. 结构清晰，用 Markdown 格式

Gemini 官网 `gemini.google.com` 在中国访问会被重定向或直接拦截。原因：

- 即使能打开页面，API 调用也需要海外 IP

- Google 生态深度整合（Gmail/Drive/YouTube/Calendar）

## 七、FAQ

**Q：Gemini 免费版和付费版差距有多大？**
A：日常对话两者差别不大。差距主要在：Advanced 可以用 Deep Research（深度研究模式）、1.5 Ultra 模型、以及更高的并发限制。普通用户免费版够用。

**Q：Gemini 能替代 ChatGPT 吗？**
A：日常对话可以。但编程、深度推理、创意写作，ChatGPT/Claude 仍然更强。建议三者结合使用。

**Q：手机怎么用 Gemini？**
A：Google AI Assistant（内置在 Android 手机）或 Gemini App（iOS 可用）。手机版和网页版功能基本一致。

**Q：Google 账号注册收不到验证码怎么办？**
A：用虚拟号码服务（如 TextNow、Google Voice）接收验证码。部分网络工具注册平台也提供临时号码服务。
