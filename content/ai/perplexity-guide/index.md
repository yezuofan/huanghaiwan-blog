## 一、Perplexity 是什么？

Perplexity 是一个**AI搜索引擎**，但它和 Google/Bing 有本质区别：

| 对比维度 | Google/Bing | ChatGPT | Perplexity |
|:--------|:-----------|:--------|:----------|
| 回答方式 | 给链接列表 | 纯文字回答（可能编造） | **文字回答 + 原文链接** |
| 信息时效 | 实时 | 训练截止日 | **实时联网** |
| 引用来源 | 无 | 无 | **每一句话都有出处** |
| 适合场景 | 简单事实查询 | 创作/推理 | **研究/查资料/写报告** |

简单说：**Perplexity = 搜索引擎 + AI 回答 + 引用来源**。

### 它的核心能力

1. **实时联网搜索**：能搜索最新新闻、股票、学术论文，不是基于训练数据的猜测
2. **带来源的回答**：每句话后面都有 `[1]` `[2]` 标注，点击可以直接跳转到原文
3. **多角度研究**：你问一个问题，它会自动搜索多个来源，给出综合观点
4. **学术友好**：能直接引用 DOI 链接，方便写论文时溯源

## 五、Perplexity 正确用法（提示词技巧）

很多人用 Perplexity 的方式不对，导致效果不如预期。以下是正确打开方式：

### 1. 把它当研究助理，而不是问答机器

❌ 错误问法：
```
"量子计算是什么"
```
→ 给你一段笼统的解释，和 Wikipedia 差不多。

✅ 正确问法：
```
"量子计算最新进展2026，有哪些突破性研究？请给出论文链接"
```
→ 给你具体的研究成果、论文名称和链接。

### 2. 追问深挖（最有价值的功能）

Perplexity 支持多轮对话。当你得到一个答案后，可以追问：

```
追问1：我之前问的量子计算研究中，哪一个最有商业化前景？
追问2：能给我找一下这个研究的中国团队吗？
追问3：对比一下中美在这个领域的投资规模差异
```

这种追问能力是 Google 做不到的——它会记住上下文，越问越深。

### 3. 用 Focus 模式精准搜索

Perplexity 有 5 种 Focus 模式，针对不同场景优化：

| Focus模式 | 适用场景 | 说明 |
|:---------|:--------|:-----|
| **全网搜索** | 通用问题 | 默认模式，搜索全网 |
| **学术搜索** | 写论文、查研究 | 优先搜索学术论文、DOI |
| **YouTube** | 找视频教程 | 搜索视频内容 |
| **Reddit** | 找真实用户讨论 | 搜用户真实评价 |
| **Wolfram** | 数学/数据计算 | 精确计算 |

**用法：** 在搜索框下方有一个 Focus 按钮，点击切换。

### 4. 收藏线程（相当于浏览器书签）

每个搜索会话会自动保存，你可以：
- 点击左侧栏的 Library 查看历史记录
- 把重要的研究线程收藏，方便以后查看
- 支持分享链接给同事

## 八、常见问题 FAQ

### Q1：Perplexity 是免费的吗？

是的，有免费版。每天 300 次搜索，足够普通人用。Pro 版 $20/月，主要是更多次数和更高模型选择。

### Q2：Perplexity 的回答可信吗？

比 ChatGPT 更可信，因为每句话都有来源链接。但仍然需要自己判断来源的质量——来自 Reddit 帖子和来自学术论文的可信度是不同的。

### Q3：国内用 Perplexity 违法吗？

Perplexity 本身是合法工具，国内用它查阅技术资料、学术信息、新闻没有法律问题。但通过非常规手段访问境外网站的法律边界较为模糊，建议了解当地规定。

### Q4：Perplexity 能用中文搜索吗？

可以，搜索中文问题效果不错，但部分来源以英文为主。如果想搜索中文内容，可以加"中国"、"中文"等限定词。

### Q5：Perplexity 和 ChatGPT Plus 该买哪个？

如果你主要用 AI **查资料、做研究**，买 Perplexity Pro。
如果你主要用 AI **写代码、创作、对话**，买 ChatGPT Plus。
两个不冲突，有预算可以都买，月费合计约 $40（≈ ¥290）。
