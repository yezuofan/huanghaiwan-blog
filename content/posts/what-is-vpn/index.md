> VPN（Virtual Private Network）即虚拟专用网络，最初用于企业远程办公，后来被广泛用于突破网络限制访问国际互联网。在中国语境下，"机场"是对 VPN 服务的俗称，因为 logo 通常是✈️的图案。

**机场 = 提供翻墙服务的供应商**

用户通过订阅机场的服务，获得节点（服务器）信息，然后通过客户端软件连接。

---

## 机场 vs 自建

| 对比 | 机场 | 自建 |
|------|------|------|
| 价格 | ¥10-50/月 | ¥5-20/月（VPS费用） |
| 难度 | 简单，订阅即用 | 需要技术知识 |
| 速度 | 取决于机场线路质量 | 取决于VPS配置和线路 |
| 稳定性 | 好的机场很稳定 | 需要自己维护 |
| 防封 | 机场一般有屏蔽对策 | 自建需要配置伪装方案 |

## 常用协议

### VMess
- 老牌协议，兼容性好
- 原生支持 XRay/V2Ray
- 被识别概率较高

### VLESS
- 更轻量，性能更好
- 配合 Reality 伪装是目前最强防封方案
- 推荐使用

### Trojan
- 基于 HTTPS 协议，伪装性强
- 配合xtls效果较好

### WireGuard
- 新一代协议，极简高速
- 对等协议，非 C/S 架构
- 目前抗封锁能力中等

## 常见名词解释

| 术语 | 解释 |
|------|------|
| 节点 | 提供服务的服务器（香港/日本/美国等） |
| 订阅 | 一串URL，包含所有节点信息，导入客户端自动更新 |
| 入口 | 用户访问机场服务器的入口线路（IEPL专线质量更好） |
| 落地 | 机场服务器最终访问目标网站的服务器 |
| 跳板 | 入口到落地之间的中转 |
| UDP | 支持游戏/语音等低延迟应用 |
| 解锁 | 能够访问特定服务（如 Netflix） |
