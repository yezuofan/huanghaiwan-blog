ChatGPT Work 是什么？为什么它比 GPT-5.6 本身更重要#

GPT-5.6 是更好的模型，但 ChatGPT Work 是更好的产品。

过去使用 AI 的典型流程是：在 ChatGPT 聊天 → 复制到 Office/WPS → 手动排版 → 调整格式。ChatGPT Work 把这几步合并成了一步——你给它一个需求和数据源，它直接输出可编辑的成品。

OpenAI CEO Sam Altman 在发布时表示，ChatGPT Work 的设计目标是「让 Codex 的能力不再局限于程序员」。一个市场分析师、产品经理、甚至学生，都可以通过自然语言让 AI 完成从数据收集到最终交付物的全过程。

核心差异：

维度 传统 ChatGPT ChatGPT Work
输出形式 文本聊天 文档/表格/PPT/Web应用
上下文来源 仅聊天窗口 Slack/Gmail/Google Drive/Notion/日历/CRM
协作能力 单人对话 团队工作空间 + 共享交付物
代码能力 仅生成代码片段 完整运行、调试、部署
插件生态 有限 GPTs 统一的插件目录

核心功能详解（4 大模块）#

1️⃣ 上下文收集引擎（Context Hub）#

ChatGPT Work 能连接到你的工作应用，自动收集相关上下文。这可能是它最「魔法」的地方：

  • Slack：读取指定频道的讨论上下文
  • Gmail / Outlook：提取邮件链中的关键信息
  • Google Drive / OneDrive：关联相关文档、表格
  • Notion / Confluence：读取知识库条目
  • 日历：了解时间线和会议安排
  • CRMs（Salesforce / HubSpot）：获取客户数据

怎么用： 在 ChatGPT Work 中关联账号后，直接说「基于上周 Slack 产品和本周邮件往来，写一份8月产品路线图更新PPT」——它会自动拉取数据并生成。

2️⃣ 交付物生成（Documents, Spreadsheets, Presentations, Web Apps）#

这是 ChatGPT Work 的核心输出能力：

文档生成：

  • 支持从提示词和来源材料生成完整报告
  • 遵循复杂格式和样式要求
  • 内嵌表格、图表、引用列表

表格/电子表格：

  • 生成带公式的电子表格
  • 支持财务模型、数据透视、图表生成
  • 精准处理数学公式和业务逻辑

演示文稿（重点升级）：

  • 支持模板和参考 PPT 风格匹配
  • 从输入内容推断设计系统（布局、排版、配色、间距）
  • GPT-5.6 相比 GPT-5.5 的关键改进： 更忠实于 Slide Master 中的嵌入规则，排版一致性大幅提升
  • 一键生成完整叙事结构

Web 应用：

  • 从自然语言需求生成轻量级网页应用
  • 可直接分享、嵌入到团队空间

3️⃣ ChatGPT 托管站点（Hosted Sites）#

一个看似不起眼但可能影响深远的附带功能：ChatGPT Work 可以直接托管和分享你生成的文档/演示文稿，作为在线站点。

你生成的 PPT/文档不是下载到本地——它有一个可直接分享的在线地址，适合团队协作审阅。这让人联想到过去的 Google Slides 模式，但不同的是内容完全由 AI 生成。

The Verge 的 David Imel 评价说：「这不难想象它会发展成为完整的网站构建器——OpenAI 这是要做 Squarespace 的对手吗？」

4️⃣ Ultra 多代理模式（Pro/Enterprise）#

对于复杂项目，ChatGPT Work 支持 Ultra 模式：默认协调 4 个并行子代理分拆任务，各自完成后汇总。

这在 GPT-5.6 Sol Ultra 模式下表现尤为突出——Terminal-Bench 2.1 得分 91.9%（4 代理），远超单代理的 88.8%。在 BrowseComp 和 SEC-Bench Pro 上，多代理增益同样显著。

ChatGPT Work vs Claude Cowork：两大AI工作台对决#

ChatGPT Work 的直接竞品是 Anthropic 的 Claude Cowork（Claude + Claude Code 组合）。以下是关键对比：

对比维度 ChatGPT Work Claude Cowork
底座模型 GPT-5.6 Sol / Terra / Luna Claude Mythos 5 / Fable 5 / Sonnet 5
上下文来源 Slack, Gmail, Google Drive, Notion, 日历, CRMs GitHub, 本地文件系统, VS Code, Slack
交付物类型 文档/表格/PPT/Web应用 Code/文档/分析报告
定价起点 Plus $20/月含完整权限 Claude Pro $20/月含部分 Cowork
web端体验 ✅ ChatGPT Work Web 版可用 ✅ Claude Cowork Web 版可用
桌面端体验 ✅ Mac/Windows 原生应用 ✅ Claude 桌面端 + VS Code 扩展
免费体验 ✅ Free 用户桌面端有限可用 ❌ Cowork 需 Pro 以上
多代理 Ultra 模式（4代理并行） 同等功能（Claude Code + Agents）

我的建议： 如果你是知识工作者（写报告、做PPT、分析数据）→ ChatGPT Work 更合适，它的文档/表格/PPT生成能力更强。如果你是开发者→ 两者都可，但 Claude Code 在编码深度上仍有优势。最佳方案：两个都试用，留一个主力。

使用教程：5 步上手 ChatGPT Work#

第一步：下载桌面应用#

ChatGPT Work 的最佳体验在桌面端（Mac / Windows）。免费用户也能用：

  1. 打开 chatgpt.com 登录账号
  2. 下载桌面应用 → 安装 → 打开
  3. 在侧边栏找到 Work 标签（默认即打开）

第二步：关联工作应用#

  1. 点击设置 → Connected Apps
  2. 授权连接 Slack、Gmail、Google Drive、Notion 等服务
  3. 关键操作：只授权需要的频道/文件夹，保护隐私

第三步：下达工作指令#

ChatGPT Work 的 prompt 风格与普通 ChatGPT 不同——它更需要明确的输出格式 + 上下文来源：

❌ 低效指令：

「帮我写一份产品更新报告」

✅ 高效指令：

「基于上周 Slack #product-updates 频道的讨论，以及本周 Gmail 中与客户方的邮件往来，生成一份8月产品路线图更新PPT。使用公司模板，包含目标、进度、风险三个章节。每页一张关键图表。」

第四步：审阅与精调#

ChatGPT Work 生成的交付物可以直接在应用中预览和调整：

  • 文档：在线编辑文字、调整格式
  • 表格：修改数据、调整公式
  • PPT：切换页面、调整样式
  • Web app：实时预览修改效果

不满意？ 直接说「第三页的数据换成饼图」、「第一段缩短到200字」——它会重新生成。

第五步：分享与协作#

生成的交付物可以：

  • 直接分享链接（Hosted Sites 功能）
  • 导出为文件（.docx / .xlsx / .pptx / .html）
  • 嵌入到工作空间（SharePoint / Google Drive / Notion）

FAQ#

Q：ChatGPT Work 和 ChatGPT 有什么区别？ A：ChatGPT 是对话助手，ChatGPT Work 是从对话到交付物的工作台。它整合了 Codex 的代码执行能力和 ChatGPT 的对话能力，输出的是可直接使用的成品（文档/表格/PPT），而不是文字回复。

Q：ChatGPT Work 免费能用吗？ A：能。 免费用户通过桌面端可以有限体验 ChatGPT Work（GPT-5.6 Terra 级别）。要完整使用（Sol 模型、多代理、全平台）需要 Plus（$20/月）以上。

Q：支持中文输出吗？ A：支持。GPT-5.6 的中文能力是当前所有模型中最强的之一，生成中文文档、PPT、表格效果很好。

Q：ChatGPT Work 会取代 Codex 吗？ A：不会。OpenAI 的 Codex 工程负责人 Thibault Sottiaux 在 7 月 10 日明确表示 Codex 独立应用「会继续存在」，并预告了即将推出的更新（侧边栏聊天和项目管理）。ChatGPT Work 更像是 Codex 能力的民主化版本——让非程序员也能用 Codex 干活的入口。

Q：ChatGPT Work 的安全性和隐私怎么样？ A：ChatGPT Work 继承了 GPT-5.6 的安全体系——分层保护、实时监控、按风险级别校准访问权限。Enterprise 用户有数据隔离选项。使用 Connected Apps 时建议只授权必要的数据范围。

Q：中国用户能用 ChatGPT Work 吗？ A：可以，但需要稳定的网络环境。桌面端体验优于Web版——下载后离线基本功能也可用（需要网络连接时才会联网）。配合高质量的网络工具体验更佳。

Q：ChatGPT Work 和 Claude Cowork 哪个更好？ A：取决于你的主要工作类型。知识工作（报告/PPT/表格）→ ChatGPT Work 更强。编程开发 → Claude Cowork 的代码深度略优。两者都有免费试用，建议都体验后决定。

Q：Plus 用户每天能用 ChatGPT Work 多少次？ A：OpenAI 未公布具体限额，但表示 Plus 用户的 ChatGPT Work 使用「扩展可用」。实际使用中，常规文档/PPT生成每日数十次无压力。高频使用建议 Pro 计划。

总结#

ChatGPT Work 是 OpenAI 在 GPT-5.6 之外最重要的发布。它回答了一个核心问题：AI 不应该只能聊天，它应该能干活。

  • 从聊天到交付物的一步到位
  • 免费用户也可入门体验
  • Plus（$20/月）即可获得完整能力
  • 竞品 Claude Cowork 各有千秋，试用后选最适合的
