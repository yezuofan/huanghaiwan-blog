Fable 5 是什么？为什么它值得关注？#

Claude Fable 5 是 Anthropic 在2026年6月9日发布的创意与通用模型。与同日发布的 Cybersecurity 模型 Claude Mythos 5 使用相同的基础模型，但 Fable 5 配备了 Anthropic 历史上最严格的安全护栏，使其适合普通用户的日常使用。

Fable 5 的定位很独特：它不像 Sonnet 5 那样偏重编码和 Agent 能力（详见 Sonnet 5 指南），而是在创意写作、内容生成、文本分析和日常推理方面表现突出。

但 Fable 5 真正出圈的原因，是它经历的"一周封禁风波"。

完整时间线：发布 → 封禁 → 回归#

Fable 5 的短暂历史本身就是一出现代 AI 治理的缩影：

日期 事件
6月9日 Anthropic 发布 Fable 5 和 Mythos 5。Fable 为安全版面向公众，Mythos 仅限 Glasswing 合作伙伴
6月12日 美国商务部对 Mythos 5 和 Fable 5 实施出口管制，禁止非美国公民访问
6月12日 Anthropic 立即暂停两个模型的全球访问
6月26日 Mythos 5 对美国组织恢复访问
6月30日 美国商务部解除出口管制
7月1日 Fable 5 全球恢复访问，用户可在 Claude.ai、Claude Code、Claude Cowork 中使用

封禁导火索： 亚马逊研究人员发现了一种绕过 Fable 5 安全护栏的方法——通过特定提示词让模型识别软件漏洞，并展示利用代码。但 Anthropic 随后的测试表明，几乎所有主流模型（包括 Claude Haiku 4.5、Opus 4.8、GPT-5.5 等）都能复现同样的能力——Fable 5 并未表现出任何"Mythos 级别"的独特网络攻击能力。

Anthropic 迅速对安全分类器进行了升级，将报告中描述的绕过技术在 99%+ 的情况下成功拦截，最终促成美国政府解除管制。

Fable 5 核心能力#

创意与内容创作（最强项）#

Fable 5 在以下任务上表现出色于其他 Claude 模型：

  • 创意写作：小说、剧本、营销文案、广告创意
  • 内容改写与润色：多种风格切换，从专业到亲和自如
  • 长文生成：能维持长篇幅内容的前后一致性和质量
  • 翻译与本地化：多语言翻译质量优异
  • 头脑风暴：创意方案发散和收敛能力强

通用推理能力#

  • 日常问答与知识检索
  • 复杂文档理解与摘要
  • 多步骤推理与决策支持
  • 文本分析与模式识别

使用教程：4种方式用上 Fable 5#

方式一：Claude.ai Web 端（最简单）#

登录 claude.ai，在模型选择器中选择 Fable 即可开始使用。Pro 订阅用户可直接访问。

适合：日常写作、翻译、头脑风暴。

方式二：Claude Code 终端#

安装 Claude Code CLI，在配置中选择 Fable 模型：

# 安装 Claude Code（如已安装可跳过）
npm install -g @anthropic/claude-code

# 启动 Claude Code 并选择 Fable 模型
claude code --model claude-fable-5

适合：开发者的创意编程辅助。

方式三：Claude Cowork 桌面应用#

安装 Claude Cowork（Mac/Windows），在模型选项中选择 Fable。Cowork 可以访问本地文件系统，让 Fable 处理本地文档的创意编辑任务。

适合：需要与本地文件协作的文案创作、报告润色。

方式四：API 调用（开发者）#

import anthropic

client = anthropic.Anthropic(api_key="your-api-key")

message = client.messages.create(
    model="claude-fable-5",
    max_tokens=4096,
    messages=[
        {"role": "user", "content": "写一篇关于人工智能未来发展的短篇科幻小说，500字左右"}
    ]
)
print(message.content[0].text)

适合：集成到自有应用的开发者和企业用户。

FAQ#

Fable 5 和 Sonnet 5 有什么区别？#

Fable 5 偏重创意写作、内容生成和文本推理；Sonnet 5 偏重编码、Agent 工具使用和自动化任务。简单来说：Fable 5 适合写内容，Sonnet 5 适合写代码。

Fable 5 真的被封禁过吗？现在安全吗？#

是的，6月12日至7月1日期间被暂停访问。但 Anthropic 与美国政府紧密合作，升级了安全分类器（>99%拦截率），已获商务部批准恢复。目前是安全可用的。

Fable 5 在中国能用吗？#

Fable 5 回归后是全球可用的，但访问 Claude.ai 本身需要合适的网络环境。中国用户需要通过以下方式使用：

  • 方案A：使用 claude.ai Web 端（需网络工具）
  • 方案B：通过第三方聚合平台访问 API（部分平台已集成）
  • 方案C：企业通过 AWS/GCP 等云服务接入

免费用户能用 Fable 5 吗？#

Fable 5 是 Pro 及以上方案 才支持。Free 方案只能使用 Sonnet 和 Haiku 系列模型。

Fable 5 编程能力怎么样？#

Fable 5 可以编程，但安全分类器在涉及网络安全相关代码时可能误拦截。如果你主要做开发，推荐使用 Sonnet 5。

有没有使用额度限制？#

7月7日前，Fable 5 最多占用用户周用量的50%；之后全额计入使用额度。Pro 方案通常每周有数万条消息的额度，日常使用绰绰有余。
