更新：2026-07-18 — Kimi K3 于 7 月 16 日发布，开源权重预计 7 月 27 日放出。Gizmodo 称其为"中国投向美国前沿AI公司的又一颗炸弹"。

2026 年 7 月 16 日，阿里巴巴投资的月之暗面（Moonshot AI） 正式发布了 Kimi K3——一个拥有 2.8 万亿参数的混合专家模型（MoE），一经面世就在 Artificial Analysis 排行榜跃居第 3，仅次于 Anthropic 的 Claude Fable 5 和 OpenAI 的 GPT-5.6 Sol。

更令人震惊的是，它在 Arena.ai 的前端开发排行榜上直接登顶，超越了包括 Claude Fable 5 在内的所有模型。而且 Moonshot 承诺将在 7 月 27 日前开源全部权重——这将是史上最大的开源模型。

对于中国用户来说，Kimi 还有一个绝对优势：不需要任何网络工具，直接在 kimi.com 就能用。

本文为你全面解读 Kimi K3 的能力、定价、使用方式，以及它为什么是美国 AI 圈最关注的中国模型之一。

Kimi K3 是什么？为什么重要？#

Kimi 系列是 Moonshot AI 从 2023 年开始推出的国产大模型家族。从最初的 128K 上下文窗口，到 2025 年 7 月开源的 Kimi K2（1T 参数），再到 2026 年初的 K2.5/K2.6/K2.7，Moonshot 一直保持每季度一次重大迭代的节奏。

Kimi K3 是这个家族的巅峰之作：

  • 规格碾压：2.8T 参数，业界最大的开源级模型（对比：DeepSeek V4 Pro 为 1.6T 参数）
  • 架构革新：基于 Kimi Delta Attention（KDA） + Attention Residuals（AttnRes） — 两项自研架构改进，据称比 K2 实现约 2.5 倍的缩放效率提升
  • MoE 激活：896 个专家中激活 16 个，配合 Stable LatentMoE 框架实现高效推理
  • 原生多模态：支持文本、图像、视频输入，同一架构下理解多重模态
  • 100 万 token 上下文：可一次处理整本《三体》三部曲级别的内容

为什么被称为"重磅炸弹"？#

Gizmodo 在报道中这样描述：“这正是 2025 年初 DeepSeek R1 发布时那种冲击感的复刻。“Arena.ai 的 CEO Anastasios Angelopoulos 更直言 K3 是”今年可能最重要的单一发布"，以及”开源中国模型超越美国模型的那一刻"。

背后的关键背景是中美 AI 竞赛的白热化——Anthropic 刚刚指控 Moonshot 等中国公司通过"蒸馏"技术非法提取 Claude 能力，而白宫正考虑进一步收紧对华芯片出口。在这个节骨眼上发布一个逼近前沿水平的开源中国模型，政治和技术意义都非同寻常。

核心能力评测#

1. 编码能力 💻#

Kimi K3 在编码方面的表现是其最大的亮点。Moonshot 官方展示了几个惊人的案例：

能力维度 Kimi K3 竞品对比
GPU 内核优化 24 小时内优化多个内核，H200 上 vs Fable 5 有竞争力 胜 Opus 4.8、GPT-5.6 Sol、GPT-5.5
编译器开发 从零构建 MiniTriton 编译器（类 Triton MLIR 堆栈） 部分基准超 Triton 官方
芯片设计 48 小时内自主设计芯片，100MHz 时序闭合 首创"模型为模型造芯片"
科学研究复现 2 小时完成研究者 1-2 周的工作（天体物理） 跨 20+ 论文，3000+ 行 Python
前端开发 Arena.ai 排行榜 #1 超越 Claude Fable 5

特别值得一提的是**“MiniTriton 编译器”案例：K3 从零开始构建了一个完整的 GPU 编程系统，包含自己的 IR 层、优化 passes、PTX 代码生成管线，甚至在 nanoGPT 训练上实现了稳定收敛——这已经不只是"会写代码"了，而是具备系统工程能力**。

2. 知识工作与 Agent 能力 🧠#

Artificial Analysis 的独立评测显示，K3 在长程知识工作评估中达到了 1547 的 Elo 评分，比 K2.6 提高了 +732 分——差距相当于两代模型迭代。在排名上仅次于 Claude Fable 5。

Agent 能力方面，K3 的几个亮点功能：

  • Kimi Work 智能工作台：支持 Widgets（交互式组件）+ Dashboard（持久化视图），可完成 120+ 轮递归自改进的行业研究
  • Kimi Researcher：自主研究 Agent，可进行 20+ 并发子 Agent 的研究工作
  • OK Computer：从简单提示构建多页网站和可编辑幻灯片
  • Kimi Code：终端化的 Agent 编程助手

3. 多模态能力 👁️#

K3 的原生多模态架构使它在以下场景表现出色：

  • 图像理解：Simon Willison 测试显示，它能从渲染的 SVG 图片生成非常准确的 alt 文本描述
  • 视频编辑：能从 56 个源剪辑中自主剪辑出节奏同步的短视频，完成专业编辑级别的工作
  • 3D 推理：结合编码能力，可将概念/图像/视频转化为可玩的交互式 3D 体验

怎么用 Kimi K3？四种方式#

方式一：Web 端（最简单）🌐#

直接访问 kimi.com，注册账号后即可使用。Kimi K3 已在主站上线，默认使用 max thinking 模式。

适合人群： 所有用户，零配置即可开始。

方式二：Kimi Work 桌面端（最强功能）💼#

下载 Kimi Work 桌面 App（v3.1.0+），支持 Windows 和 Apple Silicon Mac。Kimi Work 集成了：

  • Agent 自动完成任务
  • Deep Research 深度研究
  • Dashboard 仪表盘定时任务
  • Swarm 多 Agent 协同

适合人群： 需要深度 Agent 能力的研究者和创作者。

方式三：Kimi Code 终端编程助手 ⌨️#

终端运行 kimi code 后选择 Kimi K3（/model 命令切换）。这是 Moonshot 对标 Claude Code/Codex 的编程 Agent。

# 安装 Kimi CLI
npm install -g @kimi/cli

# 启动 Kimi Code
kimi code

# 在会话中切换到 K3
/model kimi-k3

# 使用 K3 编程
> 帮我优化这个数据库查询性能

适合人群： 开发者、需要 AI 编程助手的人群。

方式四：API 接入（开发者）🔌#

通过 platform.kimi.ai 接入 API：

# 使用 OpenAI 兼容的 SDK
from openai import OpenAI

client = OpenAI(
    api_key="你的Kimi API密钥",
    base_url="https://api.moonshot.ai/v1"
)

response = client.chat.completions.create(
    model="kimi-k3",
    messages=[
        {"role": "user", "content": "用 Python 写一个网络爬虫"}
    ]
)
print(response.choices[0].message.content)

适合人群： 开发者，需要集成到自己的应用或工作流中。

❓ 常见问题#

Q1：Kimi K3 免费版够用吗？#

免费版（Adagio）提供基础对话功能，有速率限制。日常问答、中文写作、轻度编程完全够用。需要 Deep Research、Agent 功能或更高并发需订阅。

Q2：开源权重什么时候放？#

官方承诺 2026 年 7 月 27 日之前发布。届时可在 Hugging Face 获取全部 2.8T 参数权重。

Q3：Kimi K3 和 DeepSeek V4 Pro 比怎么样？#

K3 定位更高（前沿性能），DeepSeek 定位性价比。K3 在 DeepSWE/BrowseComp 等基准上远超 DeepSeek，但价格也贵 4-5 倍。两者并不直接竞争——K3 对标 Fable 5，DeepSeek 对标平价开源。

Q4：K3 能本地运行吗？#

不能。2.8T 参数至少需要 64+ 加速器的超节点配置。Moonshot 推荐用超节点推理。个人用户只能通过云端访问。

Q5：K3 支持中文吗？#

支持，而且是原生中文模型。 Moonshot 是中国公司，Kimi K3 的中文能力在所有媲美它的前沿模型中是最强的。

Q6：Kimi K3 的 100 万 token 上下文能做什么？#

可以一次性处理：整本《三体》三部曲、完整代码仓库（如小型开源项目）、全年财报 PDF、数小时的会议转录。对于研究员和开发者来说非常实用。

Q7：和 K2.6 比提升大吗？#

非常大。AA 长程知识工作 Elo 从 ~815 提升到 1547（+732 分）。Simom Willison 的"鹈鹕骑自行车"SVG 测试也显示显著提升。可以理解为跨了一到两代的进步。

Q8：蒸馏争议对 K3 有影响吗？#

Anthropic 此前指控 Moonshot 等中国公司通过蒸馏技术提取 Claude 能力。K3 的技术架构（KDA + AttnRes）是完全自研的，但也有观点认为中国开源模型通过蒸馏加速追赶。对普通用户来说，模型质量和服务稳定性才是关键，这个争议更多是行业层面的讨论。

总结与推荐#

Kimi K3 的发布意义重大：这是第一个在多项基准上逼近美国前沿模型的中国开源模型。对于中国用户来说，这意味着无需网络工具就能体验到世界顶级的 AI 能力。

谁适合用 Kimi K3：

  • 🇨🇳 不愿/不能配置网络工具的中国用户
  • 💻 需要强大编码 Agent 的开发者
  • 📝 需要长上下文处理的研究者和学生
  • 🤖 想探索最前沿 AI 能力的技术爱好者

谁可能更适合其他方案：

  • 需要 $10 以下平价 API → DeepSeek V4 Pro
  • 追求绝对性能极限 → Claude Fable 5（需网络工具）
  • 只需日常问答 → 免费版 Kimi K3 已足够
